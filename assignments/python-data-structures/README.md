# 📘 Assignment: Python Data Structures

## 🎯 Objective

Practice using core Python data structures by organizing, updating, and analyzing a small classroom dataset with lists, dictionaries, sets, and tuples.

## 📝 Tasks

### 🛠️ Build and Organize Student Data

#### Description
Create a program that stores student information and quiz scores. Use different data structures for different needs: a list for ordered scores, a dictionary for student records, and tuples for fixed profile data.

#### Requirements
Completed program should:

- Create a dictionary where each key is a student name and each value includes that student's scores
- Use a list to store quiz scores for each student in order
- Use a tuple to store fixed student profile data (for example, student ID and grade level)
- Print the organized data in a readable format


### 🛠️ Analyze and Clean the Dataset

#### Description
Extend your program to compute summary results and identify unique skills students are practicing.

#### Requirements
Completed program should:

- Calculate and print each student's average quiz score
- Find and print the highest and lowest quiz scores across all students
- Use a set to track unique topics covered (for example, loops, conditionals, functions)
- Update one student's data and show the result after the update
